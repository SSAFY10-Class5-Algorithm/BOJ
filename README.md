# BOJ 백준 문제 풀이
## Git Commit Message

| _Type_          | _Subject_                    |
| :-------------- | :--------------------------- |
| **[lang name]** | 문제 해결                    |
| **[fix]**       | 수정                         |
| **[docs]**      | 문서 (문서 추가, 수정, 삭제) |
| **[chore]**     | 빌드, 설정 파일              |

ex) [Type] #이슈번호 커밋메세지 `git commit -m '[python] #12 백준 1237'`

## Git Branch

| 브랜치명                                     | 작업 내용        |
| :------------------------------------------- | :--------------- |
| **main**                                     | 공용             |
| **algo/본인아이디/작업구분(ex.날짜,작업명)** | 문제 해결 브랜치 |

## File Tree

/ **Problem ID** / **이름(영어로)** / **solve file, problem info file**
